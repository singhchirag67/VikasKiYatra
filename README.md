# Vikas Ki Yatra - Premium Creator Portfolio

A premium, 3D-interactive, and highly responsive single-page portfolio built for the content creator "Vikas Ki Yatra" (focusing on Cricket, Poetry, and Current Affairs). 

This website features advanced UI/UX elements like scroll-triggered animations, 3D hover effects, a custom cursor, and dynamic typing to create a cinematic digital experience.

## 🤖 Built with AI
**Disclaimer:** This entire project was generated using **Google's Gemini Pro**. 

It was built strictly for **educational purposes and experimentation**. The goal was to test the limits of AI in generating complex frontend code, specifically focusing on mathematical 3D CSS transforms, JavaScript Intersection Observers, and modern Tailwind CSS design patterns without using heavy external libraries.

## ✨ Key Features
* **Advanced Scroll Reveals:** Elements dynamically slide and fade into view as the user scrolls, utilizing a custom JavaScript `IntersectionObserver`.
* **3D Interactive Cards:** Desktop users experience a responsive 3D tilt effect on cards that react to mouse movement.
* **Responsive Design:** Fully optimized for mobile devices. Heavy 3D calculations and custom cursors are automatically disabled on touch screens to ensure buttery-smooth performance.
* **Custom UI Elements:** Features a custom gold tracking cursor and a dynamic typewriter effect on load.
* **Horizontal Cinema Slider:** A smooth, swipeable carousel to showcase YouTube Shorts natively on the page.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **Tailwind CSS:** For rapid, premium styling and responsive utility classes.
* **Vanilla JavaScript:** For DOM manipulation, scroll observers, custom cursors, and 3D math calculations (Zero external JS libraries used).
* **FontAwesome:** For lightweight, scalable icons.

## 🚀 How to Run Locally
1. Clone this repository to your local machine.
2. Open the project folder.
3. Double-click the `index.html` file to run it directly in any modern web browser. No server setup is required.
